# Min egna Miniräknare

Enkel Python-miniräknare för DevOps 1.

## Vad applikationen gör
En kommandorads-miniräknare som kan **addera**, **subtrahera**, **multiplicera** och **dividera** två tal. Innehåller även automatiserade tester.
